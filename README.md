# Data Structures

A collection of fundamental data structures implemented from scratch 
in Python, built to deepen understanding of how these structures work 
under the hood.

## Technologies Used

- Python

## Data Structures Implemented

- Linked List
- Doubly Linked List
- Stack
- Deque
- TreeSet
- TreeMap
- HashSet
- HashMap
- Graphs

## My Contribution

Solo personal project built independently outside of coursework.

## Lessons Learned

- Applied object-oriented programming principles throughout each implementation
- Worked with pointers, references, and dereferencing concepts in Python
- Implemented pre-order, in-order, and post-order tree traversals
- Gained a deeper understanding of hashing and collision handling
- Reinforced core OOP concepts such as encapsulation and composition
